# Changelog

## [2.0.0] - 2024-07-15

### Added
- Kompletna dokumentacja projektowa (README, INSTALL, USAGE, TECHNICAL)
- MIT License
- GitHub Actions CI/CD workflow
- Issue templates (bug report, feature request)
- Rozszerzone komentarze w kodzie Arduino
- Schemat elektryczny w dokumentacji
- Przykłady rozszerzeń

### Improved
- Refactoring kodu Arduino
- Lepsze komentarze i dokumentacja
- Stałe zdefiniowane na górze
- Jasna struktura kodu
- Lepsza komunikacja z GRBL

### Fixed
- Poprawiono błąd w oryginalnym kodzie (`arduino1Serial` → `Serial`)
- Usunięto duplikację znaku `;`
- Usunięto zbędny include `<SoftwareSerial.h>`

## [1.0.0] - 2023-04-01

### Added
- Inicjalne wdrożenie projektu
- Podstawowy kod sterowania przełącznikiem 3-pozycyjnego
- Komunikacja szeregowa z GRBL
- Obsługa pinów 2 i 3
- Debouncing z opóźnieniem 100ms
